# 🔍 Lights Out - Análise de Algoritmos de Busca

## 📌 Sobre o projeto

Este projeto tem como objetivo implementar e comparar diferentes algoritmos de busca aplicados ao problema clássico *Lights Out*.

O estudo foi desenvolvido como parte da disciplina de Análise de Algoritmos, com foco na avaliação prática de desempenho e eficiência.

---

## 🧠 Algoritmos implementados

* Busca em Largura (BFS)
* Busca em Profundidade Limitada (DFS)
* Busca Gulosa (Greedy)
* A*
* Hill Climbing
* Simulated Annealing

---

## 📊 Métricas analisadas

* ⏱ Tempo de execução
* 🧠 Uso de memória
* 🔍 Número de estados explorados
* ✅ Capacidade de encontrar solução
* 🎯 Qualidade da solução (optimalidade)

---

## 📏 Tamanhos testados

* 2x2
* 3x3
* 5x5
* 7x7
* 10x10

## 🧠 Conclusão resumida

Algoritmos como BFS garantem solução ótima, porém com alto custo computacional. Já o A* apresentou melhor equilíbrio entre desempenho e eficiência, enquanto algoritmos heurísticos foram mais rápidos, porém menos confiáveis.
